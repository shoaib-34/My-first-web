HTML5 Tags Reference Table

Tag Name Type Purpose / Definition
<!DOCTYPE html> Declaration Informs the web browser about the version of HTML document is running (HTML5).
<html> Structural Acts as the root container element for all other components of the HTML page.
<head> Structural Contains metadata, titles, charset declaration, which are not directly rendered visibly.
<title> Metadata Sets the text title visible on the browser tab window.
<body> Structural Encloses all the rendering content (text, tables, media) viewable to users.
<table> Layout / Data Declares a grid table container consisting of columns and rows. Used here for alignment.
<tr> Layout Defines a single individual horizontal row inside an HTML table structure.
<td> Layout Defines a standard data cell block within a table row.
<th> Layout Defines header text elements inside table tracks; text renders centered and bold by default.
<a> Inline Link Creates hyperlinks to navigate internally between pages or external URLs.
<img> Multimedia Embeds visual pictures into pages using source attributes.
<audio> Multimedia Includes native audio file players equipped with volume/play control functions.
<video> Multimedia Configures standard display zones dedicated to running native dynamic videos.
<form> Form Group Groups all interactive processing input elements into an operational layout.
<input> Form Input Configures versatile data input variants depending on 'type' configurations.
<select> Form Input Generates a clean vertical dropdown choice menu block list.
